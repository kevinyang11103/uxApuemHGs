# 旅游推荐数据可视化系统 python613

## 项目概述

本系统是一个基于Python和Django框架的旅游推荐数据可视化系统，通过机器学习算法为用户提供个性化的旅游推荐服务。

## 技术栈

- **后端：** Django
- **开发语言：** Python
- **数据库：** MySQL
- **爬虫技术：** requests
- **推荐算法：** 机器学习
- **前端：** BootStrap + ECharts

## 功能模块

### 用户系统

- 登录
- 注册
- 个人信息管理

### 数据展示

- 旅游景点信息可视化呈现

### 用户互动

- 景点评论
- 评分
- 反馈

### 算法推荐

- 基于机器学习智能推荐景点

### 数据分析

- 多维度统计旅游信息

### 可视化图表

- ECharts 图表展示

## 系统角色

- 普通用户：进行登录、注册、查看景点信息、发表评论等操作。
- 管理员：负责系统管理、用户管理和数据统计分析。

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x
- ECharts 4.x
- BootStrap 4.x

## 部署步骤

1. 环境准备：确保Python、MySQL等依赖软件已正确安装。
2. 数据库配置：按照文档配置MySQL数据库。
3. 代码部署：将项目代码部署到服务器。
4. 系统启动：按照说明启动Django服务。


- **智能推荐：** 采用机器学习算法实现个性化旅游推荐。
